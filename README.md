# PNCC implementation in Python
Based on http://www.cs.cmu.edu/~robust/Papers/OnlinePNCC_V25.pdf



## Installation:
Clone and install requirements.
```bash
cd ~
git clone https://github.com/supikiti/PNCC.git
cd PNCC
pip install -r requirements.txt
```
